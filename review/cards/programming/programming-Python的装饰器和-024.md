**Q**: Python 的装饰器和 C++ 的函数包装有什么区别？

**A**: Python 装饰器在函数定义时（编译时）通过 `@` 语法糖静态挂载；C++ 需在调用处显式包装（Lambda/函数指针），无等效语法糖
