在 C++ 中，处理图结构时，通常需要定义图的表示方式，并围绕该表示实现一系列操作。以下是常见的图操作框架：

### 1. 图的表示

#### 邻接矩阵

- **定义**：使用二维数组来表示图，数组 `adjMatrix[i][j]` 的值表示顶点 `i` 和顶点 `j` 之间的关系（例如权重，如果是无权图则用 `1` 表示有边，`0` 表示无边）。
- **适用场景**：适用于顶点数量相对较少且边比较密集的图，因为邻接矩阵的空间复杂度为 O(V2)，其中 `V` 是顶点的数量。

#### 邻接表

- **定义**：使用一个数组，数组的每个元素是一个链表（或 `std::vector`），存储与该顶点相连的所有顶点及其相关信息（如边的权重）。例如，`adjList[i]` 包含与顶点 `i` 相邻的所有顶点及其边的属性。
- **适用场景**：适用于边比较稀疏的图，空间复杂度为 O(V+E)，其中 `E` 是边的数量。

### 2. 图的创建

- **初始化顶点和边**：
    
    - **对于邻接矩阵**：初始化二维数组，根据图的类型（有向图、无向图、加权图等）设置数组元素的值。例如，对于无向无权图，若顶点 `i` 和顶点 `j` 之间有边，则 `adjMatrix[i][j] = adjMatrix[j][i] = 1`。
    - **对于邻接表**：初始化数组，并为每个顶点创建空的链表或 `std::vector`。然后根据图的边信息，将相邻顶点添加到相应顶点的链表或 `std::vector` 中。
    

### 3. 添加和删除边

- **添加边**：
    
    - **邻接矩阵**：如果是无向图，`adjMatrix[i][j] = adjMatrix[j][i] = weight`（`weight` 为边的权重，若为无权图则 `weight = 1`）；如果是有向图，`adjMatrix[i][j] = weight`。
    - **邻接表**：在 `adjList[i]` 中添加顶点 `j` 及其权重信息（若有权重），对于无向图，还需在 `adjList[j]` 中添加顶点 `i` 及其权重信息。
    
- **删除边**：
    
    - **邻接矩阵**：对于无向图，`adjMatrix[i][j] = adjMatrix[j][i] = 0`；对于有向图，`adjMatrix[i][j] = 0`。
    - **邻接表**：在 `adjList[i]` 中删除顶点 `j` 的相关信息，对于无向图，还需在 `adjList[j]` 中删除顶点 `i` 的相关信息。
    

### 4. 图的遍历

#### 深度优先搜索（DFS）

- **递归实现**：
    
    - 从起始顶点开始，标记该顶点为已访问。
    - 对该顶点的所有未访问邻接顶点递归调用 DFS 函数。
    
- **非递归实现（使用栈）**：
    
    - 创建一个栈，并将起始顶点压入栈中。
    - 创建一个集合记录已访问的顶点。
    - 当栈不为空时，弹出栈顶顶点。如果该顶点未被访问，则标记为已访问，并将其所有未访问邻接顶点压入栈中。
    

#### 广度优先搜索（BFS）

- 使用队列实现：
    
    - 创建一个队列，并将起始顶点放入队列中。
    - 创建一个集合记录已访问的顶点。
    - 当队列不为空时，取出队列头部顶点。如果该顶点未被访问，则标记为已访问，并将其所有未访问邻接顶点加入队列尾部。
    

### 5. 最短路径算法

#### Dijkstra 算法

- **适用条件**：适用于带权有向图，且所有边的权重非负。
- **操作步骤**：
    
    - 初始化距离数组 `dist`，将起始顶点到自身的距离设为 `0`，到其他顶点的距离设为无穷大。
    - 创建一个优先队列，用于存储顶点及其当前的最短距离。
    - 将起始顶点及其距离 `0` 加入优先队列。
    - 当优先队列不为空时：
        
        - 取出队首顶点 `u` 及其距离 `d`。
        - 如果 `d` 大于 `dist[u]`，则跳过（说明该顶点已被更优路径更新过）。
        - 遍历 `u` 的所有邻接顶点 `v`，计算从起始顶点到 `v` 的新距离 `newDist = dist[u] + weight(u, v)`。
        - 如果 `newDist` 小于 `dist[v]`，则更新 `dist[v] = newDist`，并将 `v` 及其新距离加入优先队列。
        
    

#### Bellman - Ford 算法

- **适用条件**：适用于带权有向图，允许存在负权边，但不允许存在负权回路。
- **操作步骤**：
    
    - 初始化距离数组 `dist`，将起始顶点到自身的距离设为 `0`，到其他顶点的距离设为无穷大。
    - 对图中的每条边进行 `V - 1` 次松弛操作（`V` 是顶点数量）。对于每条边 `(u, v)`，如果 `dist[u]` 不是无穷大且 `dist[u] + weight(u, v) < dist[v]`，则更新 `dist[v] = dist[u] + weight(u, v)`。
    - 检查是否存在负权回路，再次遍历所有边，如果存在 `dist[u]` 不是无穷大且 `dist[u] + weight(u, v) < dist[v]`，则说明存在负权回路。
    

#### Floyd - Warshall 算法

- **适用条件**：适用于带权有向图，允许存在负权边，但不允许存在负权回路，可用于求任意两点之间的最短路径。
- **操作步骤**：
    
    - 初始化距离矩阵 `dist`，`dist[i][j]` 表示顶点 `i` 到顶点 `j` 的直接距离，如果 `i` 和 `j` 之间没有直接边，则设为无穷大，`dist[i][i] = 0`。
    - 对于每个中间顶点 `k`，遍历所有顶点对 `(i, j)`，如果 `dist[i][k]` 不是无穷大且 `dist[k][j]` 不是无穷大，并且 `dist[i][k] + dist[k][j] < dist[i][j]`，则更新 `dist[i][j] = dist[i][k] + dist[k][j]`。
    

### 6. 最小生成树算法

#### Prim 算法

- **适用条件**：适用于连通无向加权图。
- **操作步骤**：
    
    - 选择一个起始顶点 `s`。
    - 初始化距离数组 `key`，将所有顶点的 `key` 值设为无穷大，`key[s] = 0`。
    - 创建一个优先队列，用于存储顶点及其 `key` 值。
    - 创建一个集合记录已加入最小生成树的顶点。
    - 当优先队列不为空时：
        
        - 取出队首顶点 `u` 及其 `key` 值。
        - 将 `u` 加入最小生成树集合。
        - 遍历 `u` 的所有邻接顶点 `v`，如果 `v` 不在最小生成树集合中，并且边 `(u, v)` 的权重小于 `key[v]`，则更新 `key[v] = weight(u, v)`，并将 `v` 及其新 `key` 值加入优先队列。
        
    

#### Kruskal 算法

- **适用条件**：适用于连通无向加权图。
- **操作步骤**：
    
    - 将图的所有边按照权重从小到大排序。
    - 创建一个并查集数据结构，用于判断两个顶点是否在同一连通分量中。
    - 初始化最小生成树的边集合为空。
    - 遍历排序后的边集合，对于每条边 `(u, v)`：
        
        - 如果 `u` 和 `v` 不在同一连通分量中（通过并查集判断），则将该边加入最小生成树的边集合，并合并 `u` 和 `v` 所在的连通分量（通过并查集操作）。
        
    

### 7. 拓扑排序

- **适用条件**：适用于有向无环图（DAG）。
- **操作步骤**：
    - 计算每个顶点的入度（即指向该顶点的边的数量）。
    - 创建一个队列，将所有入度为 `0` 的顶点加入队列。
    - 创建一个结果向量，用于存储拓扑排序的结果。
    - 当队列不为空时：
        
        - 取出队列头部顶点 `u`，将其加入结果向量。
        - 遍历 `u` 的所有邻接顶点 `v`，将 `v` 的入度减 `1`。如果 `v` 的入度变为 `0`，则将 `v` 加入队列。
        
    - 如果结果向量中包含的顶点数量等于图的顶点数量，则得到的结果向量即为拓扑排序结果；否则，说明图中存在环，无法进行拓扑排序。
