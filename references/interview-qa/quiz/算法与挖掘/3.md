## 1. 手写 PCA 核心步骤（伪代码即可）

**题目**

请用伪代码实现 PCA 降维流程，要求包含：标准化、协方差矩阵、特征值分解、投影。

### **参考答案**

```python
import numpy as np

def PCA(X, n_components):
    # 1. 标准化
    X_mean = np.mean(X, axis=0)
    X_std = np.std(X, axis=0)
    X_std[X_std == 0] = 1  # 避免除0
    X_norm = (X - X_mean) / X_std

    # 2. 协方差矩阵
    cov_mat = np.cov(X_norm.T)  # shape=(d,d)

    # 3. 特征值分解
    eig_vals, eig_vecs = np.linalg.eig(cov_mat)

    # 4. 按特征值从大到小排序
    idx = np.argsort(eig_vals)[::-1]
    eig_vecs_sorted = eig_vecs[:, idx]

    # 5. 取前n_components个主成分
    top_vecs = eig_vecs_sorted[:, :n_components]

    # 6. 投影
    X_pca = X_norm.dot(top_vecs)
    return X_pca, top_vecs, X_mean, X_std
```

**面试官常追问**

- 为什么要转置再求协方差？
- 为什么用 SVD 比 eig 更稳定？
    
    答：实际工程常用 `np.linalg.svd`，数值更稳，尤其当协方差矩阵接近奇异时。

## 2. 手写 K-Means 核心逻辑（伪代码）

**题目**

实现 K-Means 聚类，包含：初始化中心、分配簇、更新中心、迭代收敛。

### **参考答案**

```python
import numpy as np

def kmeans(X, k, max_iter=100):
    # 随机选k个中心点
    np.random.seed(0)
    centers = X[np.random.choice(len(X), k, replace=False)]
    
    for _ in range(max_iter):
        # 1. 分配：每个样本到最近中心
        labels = []
        for x in X:
            dists = [np.linalg.norm(x - c) for c in centers]
            labels.append(np.argmin(dists))
        labels = np.array(labels)

        # 2. 更新中心
        new_centers = []
        for i in range(k):
            cluster = X[labels == i]
            if len(cluster) > 0:
                new_centers.append(cluster.mean(axis=0))
            else:
                new_centers.append(centers[i])  # 空簇不更新
        new_centers = np.array(new_centers)

        # 3. 收敛判断
        if np.all(centers == new_centers):
            break
        centers = new_centers
        
    return labels, centers
```

**高频考点**

- K-Means 损失函数：SSE（簇内平方和）
- 如何选 K：手肘法、轮廓系数
- 缺点：对初始点敏感、对异常值敏感

## 3. 手写 BM25 单词打分公式实现（可直接口述）

**题目**

实现 BM25 单个词的打分函数，给出可运行片段。

### **参考答案**

```python
import math

def bm25_score(tf, df, N, doc_len, avgdl, k1=1.2, b=0.75):
    # IDF
    idf = math.log((N - df + 0.5) / (df + 0.5) + 1)
    
    # TF 饱和 + 长度归一化
    numerator = tf * (k1 + 1)
    denominator = tf + k1 * (1 - b + b * doc_len / avgdl)
    tf_part = numerator / denominator
    
    return idf * tf_part
```

**关键点**

- IDF 部分加 1 防止负分
- 打分是**词独立求和**
- RAG 中一般 BM25 召回 top30~50，再和向量召回合并

## 4. 手写 XGBoost 决策树分裂增益（伪代码）

**题目**

写出 XGBoost 分裂节点时的结构分数（gain）计算逻辑。

### **参考答案**

XGBoost 分裂增益公式：
![[Pasted image 20260416095734.png]]
```python
def xgb_gain(G_L, H_L, G_R, H_R, lambd=1, gamma=0):
    left = G_L**2 / (H_L + lambd)
    right = G_R**2 / (H_R + lambd)
    total = (G_L + G_R)**2 / (H_L + H_R + lambd)
    gain = 0.5 * (left + right - total) - gamma
    return gain
```

**面试官追问**

- G、H 分别是什么？
    
    答：G = 一阶梯度 g，H = 二阶梯度 h
- γ 作用：控制树复杂度，越大越不容易分裂

## 5. 手写 RAG 混合召回（BM25 + 向量）合并逻辑

**题目**

实现 RAG 中最常用的：BM25 召回 + 向量召回 → 去重 → 合并 topK。

### **参考答案**

```python
def hybrid_recall(bm25_results, vector_results, top_k=10):
    # 结果格式: [(doc_id, score), ...]
    
    # 1. 合并并去重
    doc_dict = {}
    # BM25
    for doc_id, score in bm25_results:
        doc_dict[doc_id] = {"bm25": score, "vec": 0}
    # 向量
    for doc_id, score in vector_results:
        if doc_id in doc_dict:
            doc_dict[doc_id]["vec"] = score
        else:
            doc_dict[doc_id] = {"bm25": 0, "vec": score}
    
    # 2. 加权融合（可调节权重）
    fused = []
    for doc_id, s in doc_dict.items():
        fused_score = 0.3 * s["bm25"] + 0.7 * s["vec"]
        fused.append((doc_id, fused_score))
    
    # 3. 排序取top
    fused.sort(key=lambda x: x[1], reverse=True)
    return fused[:top_k]
```

**工程要点**

- BM25 做字面匹配，向量做语义
- 实际会先各取 top50，再融合
- 后面一般再接 CrossEncoder 重排
