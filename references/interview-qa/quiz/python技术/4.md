## 简答参考答案

- merge：按指定键（如 user_id）关联多表，支持 inner/left/right/full，适合业务表关联。
- join：按索引连接，简洁但不如 merge 灵活。
- concat：按行或列直接拼接，适合追加数据、合并同结构表。
    
    中台常用：多表宽表构建用 merge，同结构日志合并用 concat。

- 判断：duplicated ()、groupby.count ()>1、主键唯一性检查。
- 处理：
    
    1. 完全重复行直接删除。
    2. 同一用户多条记录保留最新时间。
    3. 订单重复按订单号去重，保留有效状态。
    4. 日志重复按唯一事件 ID 去重。

# 三、Numpy 编程题（1 题 ×12 分 = 12 分）

生成形状 (200, 6) 的标准正态分布随机数组，完成：

1. 按列求均值、方差
2. 对每一列做标准化
3. 将绝对值大于 2 的值替换为该列中位数
4. 输出处理后数组的形状与均值

### 答案

```python
import numpy as np
np.random.seed(42)

# 1
data = np.random.randn(200, 6)
mean = data.mean(axis=0)
var = data.var(axis=0)

# 2
std = data.std(axis=0)
data_std = (data - mean) / std

# 3
for i in range(6):
    col = data_std[:, i]
    mask = np.abs(col) > 2
    median = np.median(col)
    data_std[mask, i] = median

# 4
print(data_std.shape)
print(data_std.mean())
```

# 四、Pandas 数据清洗 & 质量治理（1 题 ×20 分 = 20 分）

给定用户数据表 df，包含字段：

user_id, phone, age, city, register_time, pay_amount

要求：

1. 删除缺失率 > 25% 的列
2. 按 user_id 去重，保留 register_time 最新的一条
3. 手机号清洗：仅保留数字，长度≠11 记为 'invalid'
4. age 异常值（<16 或>70）设为该列中位数
5. pay_amount<0 设为 0
6. 新增字段 data_flag：缺失字段数 = 0 为 1，否则 0

## 模拟数据 + 答案

```python
import pandas as pd
import numpy as np

df = pd.DataFrame({
    'user_id': [1,1,2,3,4,5,5],
    'phone': ['13800138000','13800138000','1234',None,'13999999999','abc123','136111122223'],
    'age': [18,18,80,None,30,10,35],
    'city': ['bj','bj','sh',None,'gz','sz','sz'],
    'register_time': pd.date_range('2024-01-01', periods=7),
    'pay_amount': [100,100,-50,200,None,300,500]
})

# 1
missing = df.isnull().sum() / len(df)
df = df.drop(columns=missing[missing>0.25].index, errors='ignore')

# 2
df = df.sort_values('register_time').drop_duplicates('user_id', keep='last')

# 3
df['phone'] = df['phone'].astype(str).str.replace(r'\D', '', regex=True)
df['phone'] = df['phone'].where(df['phone'].str.len()==11, 'invalid')

# 4
med_age = df['age'].median()
df['age'] = df['age'].clip(16,70).fillna(med_age)

# 5
df['pay_amount'] = df['pay_amount'].fillna(0)
df.loc[df['pay_amount']<0, 'pay_amount'] = 0

# 6
df['data_flag'] = (df.isnull().sum(axis=1)==0).astype(int)
```

# 五、数据中台指标聚合（1 题 ×14 分 = 14 分）

两张表：

user: user_id, channel, register_dt

order: order_id, user_id, pay_amt, pay_dt

要求：

1. 左连接，保留所有用户
2. 统计每个用户：订单数、总支付金额、首支付时间
3. 筛选 2024 年注册且渠道为 'ad' 的付费用户
4. 计算各渠道的付费转化率
### 答案

```python
import pandas as pd

user = pd.DataFrame({
    'user_id': [1,2,3,4,5],
    'channel': ['ad','natural','ad','natural','ad'],
    'register_dt': pd.to_datetime([
        '2024-01-01','2023-12-01','2024-02-01','2024-03-01','2023-11-01'
    ])
})

order = pd.DataFrame({
    'order_id': [101,102,103,104],
    'user_id': [1,1,3,3],
    'pay_amt': [100,200,150,300],
    'pay_dt': pd.to_datetime(['2024-01-02','2024-01-03','2024-02-02','2024-02-05'])
})

# 1
df = pd.merge(user, order, on='user_id', how='left')

# 2
agg_df = df.groupby('user_id').agg(
    order_cnt=('order_id','count'),
    total_amt=('pay_amt','sum'),
    first_pay=('pay_dt','min')
).reset_index()

user = user.merge(agg_df, on='user_id', how='left')

# 3
cond1 = user['register_dt'] >= '2024-01-01'
cond2 = user['channel'] == 'ad'
cond3 = user['order_cnt'] > 0
result = user[cond1 & cond2 & cond3]

# 4
user['is_pay'] = user['order_cnt']>0
conv_rate = user.groupby('channel')['is_pay'].mean()
print(conv_rate)
```

# 六、特征工程 & 建模基础（1 题 ×14 分 = 14 分）

对用户特征完成标准特征工程：

1. 分类型特征 OneHot 编码（drop='first'）
2. 数值型特征标准化
3. 方差过滤剔除低方差特征
4. 分层划分训练集测试集
### 答案
```python
import pandas as pd
from sklearn.preprocessing import OneHotEncoder, StandardScaler
from sklearn.feature_selection import VarianceThreshold
from sklearn.model_selection import train_test_split

df = pd.DataFrame({
    'age': [22,31,28,35,40],
    'salary': [5000,9000,7500,12000,14000],
    'gender': ['m','f','m','f','m'],
    'city': ['bj','sh','bj','sz','sh'],
    'churn': [0,1,0,1,0]
})

X = df.drop('churn', axis=1)
y = df['churn']

cat_cols = ['gender','city']
num_cols = ['age','salary']

# 1
ohe = OneHotEncoder(drop='first', sparse_output=False)
cat_feat = ohe.fit_transform(X[cat_cols])
cat_df = pd.DataFrame(cat_feat, columns=ohe.get_feature_names_out())

# 2
scaler = StandardScaler()
num_feat = scaler.fit_transform(X[num_cols])
num_df = pd.DataFrame(num_feat, columns=num_cols)

X_all = pd.concat([num_df, cat_df], axis=1)

# 3
sel = VarianceThreshold(0.01)
X_selected = sel.fit_transform(X_all)

# 4
X_train,X_test,y_train,y_test = train_test_split(
    X_selected, y, test_size=0.2, random_state=42, stratify=y
)
```
