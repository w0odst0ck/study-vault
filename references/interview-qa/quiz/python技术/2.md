## 一、基础理论题（3 题，每题 8 分）

### 1. 在数据治理中，为什么经常用 `astype('category')`？和普通字符串类型相比有什么优势？

#### **参考答案**

- 减少内存占用：字符串重复值多时，category 只存储唯一值 + 索引，内存大幅降低。
- 提升分组、筛选、join 速度：category 底层是整数编码，计算更快。
- 便于数据标准管理：可强制枚举值范围，方便做数据质量校验。
- 适合性别、城市、等级、状态码等维度字段，是数据中台维度表常用优化手段。

### 2. Pandas 中 `apply`、`map`、`applymap` 的区别，分别适合什么场景？

#### **参考答案**

- `map`：仅用于 **Series**，对每个元素做映射，适合字典映射、简单函数。
- `apply`：可用于 **Series / DataFrame**，更灵活，支持复杂逻辑、多列运算。
- `applymap`：旧版用于 **DataFrame 元素级**操作（新版用 `df.map()`）。
    
    数据治理场景：字段标准化、枚举值映射多用 `map`；复杂清洗规则多用 `apply`。

### 3. 简述标准化（StandardScaler）与归一化（MinMaxScaler）的区别，数据中台建模时如何选择？

#### **参考答案**

- 标准化：(x-μ)/σ，均值 0 方差 1，受异常值影响相对小。
- 归一化：缩放到 [0,1] 或 [-1,1]，依赖极值，异常值影响大。
    
    选择：
- 树模型一般不用缩放；
- 线性模型、SVM、PCA 常用标准化；
- 图像、区间约束场景用归一化。
    
    数据治理建模优先标准化，更稳健。

## 二、Numpy 实操题（1 题，12 分）

生成形状 (1000, 5) 的随机正态分布数组，完成：

1. 按行求均值
2. 按列求中位数
3. 把每一行都归一化到 [0,1]
4. 找出所有大于 2σ 的异常值位置并标记为 1，否则 0

#### **参考答案**

```python
import numpy as np

np.random.seed(42)
data = np.random.randn(1000, 5)

# 1 行均值
row_mean = data.mean(axis=1)

# 2 列中位数
col_median = np.median(data, axis=0)

# 3 行归一化 [0,1]
row_min = data.min(axis=1, keepdims=True)
row_max = data.max(axis=1, keepdims=True)
data_norm = (data - row_min) / (row_max - row_min)

# 4 2σ异常值标记
mean = data.mean()
std = data.std()
outlier_mask = (data > mean + 2*std) | (data < mean - 2*std)
outlier_label = np.where(outlier_mask, 1, 0)
```

## 三、Pandas 数据清洗与质量治理（1 题，18 分）

给定用户表，要求实现**企业级数据质量处理**：

1. 统计缺失率，缺失率 > 30% 的列直接丢弃
2. 重复行：按 user_id 去重，保留最新 create_time
3. 手机号格式清洗：只保留数字，长度非 11 标记为异常
4. 数值型字段用中位数填充，分类型用众数填充
5. 新增一列 `data_level`：缺失字段数 = 0 为 “优秀”，≤2 为 “合格”，>2 为 “不合格”

#### **模拟数据 + 参考答案**

```python
import pandas as pd
import numpy as np

df = pd.DataFrame({
    'user_id': [1,1,2,3,4,5,5],
    'name': ['a','a','b',None,'c','d','d'],
    'phone': ['13800138000','13800138000','12345',None,'13999999999','abc123',None],
    'age': [22,22,None,35,None,28,28],
    'city': ['bj','bj','sh',None,'gz','sz','sz'],
    'create_time': pd.date_range('2024-01-01', periods=7)
})

# 1 缺失率过滤
missing = df.isnull().sum() / len(df)
drop_cols = missing[missing > 0.3].index
df = df.drop(columns=drop_cols, errors='ignore')

# 2 去重
df = df.sort_values('create_time').drop_duplicates('user_id', keep='last')

# 3 手机号清洗
df['phone'] = df['phone'].astype(str).str.replace(r'\D', '', regex=True)
df['phone_valid'] = df['phone'].str.len() == 11

# 4 填充
num_cols = df.select_dtypes(include='number').columns
df[num_cols] = df[num_cols].fillna(df[num_cols].median())

cat_cols = df.select_dtypes(include='object').columns
for c in cat_cols:
    df[c] = df[c].fillna(df[c].mode()[0])

# 5 数据等级
df['null_cnt'] = df.isnull().sum(axis=1)
df['data_level'] = df['null_cnt'].apply(
    lambda x: '优秀' if x==0 else '合格' if x<=2 else '不合格'
)
```

## 四、数据中台指标加工 & 多表关联（1 题，18 分）

两张表：

- user：user_id, register_dt, channel
- order：order_id, user_id, pay_amount, pay_dt

要求：

1. 左连接保留所有用户
2. 计算每个用户的：订单数、总支付金额、首次支付时间、末次支付时间
3. 筛选出 “2024 年注册且渠道为‘广告’且有支付” 的用户
4. 计算渠道转化率（支付用户数 / 注册用户数）

#### **参考答案**

```python
import pandas as pd

user = pd.DataFrame({
    'user_id': [1,2,3,4,5],
    'register_dt': pd.to_datetime(['2024-01-01','2023-12-01','2024-02-01','2024-03-01','2023-11-01']),
    'channel': ['广告','自然','广告','自然','广告']
})

order = pd.DataFrame({
    'order_id': [101,102,103,104],
    'user_id': [1,1,3,3],
    'pay_amount': [100,200,150,300],
    'pay_dt': pd.to_datetime(['2024-01-02','2024-01-03','2024-02-05','2024-02-06'])
})

# 1 左连接
df = pd.merge(user, order, on='user_id', how='left')

# 2 用户指标
user_metric = df.groupby('user_id').agg(
    order_cnt=('order_id','count'),
    total_amt=('pay_amount','sum'),
    first_pay=('pay_dt','min'),
    last_pay=('pay_dt','max')
).reset_index()

user = pd.merge(user, user_metric, on='user_id', how='left')

# 3 筛选
cond1 = user['register_dt'] >= '2024-01-01'
cond2 = user['channel'] == '广告'
cond3 = user['order_cnt'] > 0
result = user[cond1 & cond2 & cond3]

# 4 转化率
conv = user.assign(has_pay=user['order_cnt']>0)
conv_rate = conv.groupby('channel')['has_pay'].mean()
```

## 五、特征工程与建模基础（1 题，16 分）

对用户特征做标准特征工程流程：

1. 分类型特征做 OneHot 并避免多重共线（drop='first'）
2. 数值型特征做标准化
3. 使用方差过滤剔除低方差特征
4. 划分训练集测试集，stratify 保证标签均衡

#### **参考答案**

```python
import pandas as pd
from sklearn.preprocessing import OneHotEncoder, StandardScaler
from sklearn.feature_selection import VarianceThreshold
from sklearn.model_selection import train_test_split

df = pd.DataFrame({
    'age': [23,34,45,29,51],
    'salary': [5000,8000,12000,6500,14000],
    'gender': ['m','f','m','f','m'],
    'city': ['bj','sh','bj','sz','sh'],
    'churn': [0,1,0,1,0]
})

X = df.drop('churn', axis=1)
y = df['churn']

cat_cols = ['gender','city']
num_cols = ['age','salary']

# 1 OneHot
ohe = OneHotEncoder(drop='first', sparse_output=False)
cat_feat = ohe.fit_transform(X[cat_cols])
cat_df = pd.DataFrame(cat_feat, columns=ohe.get_feature_names_out())

# 2 标准化
scaler = StandardScaler()
num_feat = scaler.fit_transform(X[num_cols])
num_df = pd.DataFrame(num_feat, columns=num_cols)

X_all = pd.concat([num_df, cat_df], axis=1)

# 3 方差过滤
selector = VarianceThreshold(threshold=0.01)
X_selected = selector.fit_transform(X_all)

# 4 数据集划分
X_train, X_test, y_train, y_test = train_test_split(
    X_selected, y, test_size=0.2, random_state=42, stratify=y
)
```

## 六、综合场景题（数据治理 / 中台实战）（10 分）

问题：**你在数据中台建设中如何用 Python 保证数据质量？请写出可落地的流程与核心代码思路。**

**参考答案（面试可直接背诵）**

1. **数据探查**：info、describe、缺失率、唯一值分布。
2. **数据标准**：字段类型统一、枚举值标准化、时间格式统一。
3. **主键唯一性校验**：检查主键是否重复。
4. **业务规则校验**：金额 > 0、年龄合理、时间逻辑合法。
5. **质量报告输出**：输出不合格记录，用于回流治理。

#### **核心代码框架**

```python
# 质量检查
df['rule_amt'] = df['pay_amount'] > 0
df['rule_age'] = df['age'].between(16, 70)
df['rule_time'] = df['pay_dt'] > df['register_dt']

# 质量分
df['quality_score'] = df[['rule_amt','rule_age','rule_time']].all(axis=1)

# 异常数据输出
bad_data = df[~df['quality_score']]
bad_data.to_csv('data_quality_error.csv', index=False)
```
