# 华为 OD 机考特别适配建议

1. **Python 写法优先**
    
    - 不用炫技，用最直观的循环、递归、基础数据结构
    - 多用`dict`、`list`、`collections.deque`、`heapq`
    
2. **机考答题策略**
    
    - 先做简单题保分，再冲中等题
    - 不会的题写暴力解法也能拿部分分
    - 注意输入输出格式（OD 机考需要标准 IO）
    
3. **每日 3.5h 标准安排**
    
    - 0.5h：看题解 / 学思路
    - 2.0h：独立敲代码
    - 0.5h：复盘 + 整理模板
    
4. **弹性调整**
    
    - 时间多：每天多刷 1~2 题
    - 时间少：优先做标记⭐的华为高频题
