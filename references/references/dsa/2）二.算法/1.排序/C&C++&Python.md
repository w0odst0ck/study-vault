在 C++ 中，排序算法是非常重要的基础算法，以下是常见排序算法的框架：

### 1. 冒泡排序

- **基本思想**：重复地比较相邻的两个元素，如果它们的顺序错误就把它们交换过来。每一趟排序都会将当前未排序部分的最大（或最小）元素 “冒泡” 到末尾。
- **操作步骤**：
    
    - 从数组的第一个元素开始，到数组倒数第二个元素结束，进行多趟比较。
    - 在每一趟中，比较相邻的两个元素，如果顺序不对（例如升序时前一个大于后一个），则交换这两个元素。
    - 每一趟结束后，下一趟比较的范围减少一个元素，因为最后一个元素已经是当前未排序部分的最大（或最小）元素。
    

### 2. 选择排序

- **基本思想**：每一趟从待排序的数据元素中选出最小（或最大）的一个元素，存放在序列的起始位置，直到全部待排序的数据元素排完。
- **操作步骤**：
    
    - 从数组的第一个元素开始，到数组倒数第二个元素结束，进行多趟选择。
    - 在每一趟中，假设当前位置的元素为最小（或最大）元素，然后从当前位置之后的元素中找到真正的最小（或最大）元素。
    - 如果找到的最小（或最大）元素不是假设的元素，则交换这两个元素。
    

### 3. 插入排序

- **基本思想**：将一个数据插入到已经排好序的数组中的适当位置。类似于扑克牌的整理过程，手中的牌是已排序的，每次从桌上拿起一张牌，插入到手中合适的位置。
- **操作步骤**：
    
    - 从数组的第二个元素开始，将其视为待插入元素。
    - 把待插入元素与已排序部分（即其前面的元素）从后往前逐个比较。
    - 如果待插入元素小于（或大于）比较的元素，则将比较的元素往后移动一个位置，为待插入元素腾出位置。
    - 重复上述步骤，直到找到合适的位置插入待插入元素。
    

### 4. 快速排序

- **基本思想**：通过一趟排序将待排记录分割成独立的两部分，其中一部分记录的关键字均比另一部分记录的关键字小，然后分别对这两部分记录继续进行排序，以达到整个序列有序。
- **操作步骤**：
    
    - 选择一个基准元素（通常选择数组的第一个元素、最后一个元素或中间元素）。
    - 设立两个指针，一个指向数组开头（left），一个指向数组末尾（right）。
    - 从右向左移动 right 指针，找到第一个小于基准元素的元素；从左向右移动 left 指针，找到第一个大于基准元素的元素。
    - 如果 left 指针小于 right 指针，则交换这两个元素，然后继续移动指针。
    - 当 left 指针和 right 指针相遇时，将基准元素与相遇位置的元素交换，此时基准元素左边的元素都小于它，右边的元素都大于它。
    - 递归地对基准元素左边和右边的子数组进行快速排序。
    

### 5. 归并排序

- **基本思想**：采用分治法，将一个数组分成两个子数组，对这两个子数组分别进行排序，然后将排好序的子数组合并成一个最终的有序数组。
- **操作步骤**：
    
    - **分解**：将数组不断地分成两半，直到每个子数组只包含一个元素（单个元素的数组是有序的）。
    - **合并**：将两个有序的子数组合并成一个更大的有序数组。在合并过程中，比较两个子数组的当前元素，将较小（或较大）的元素放入结果数组中，然后移动相应子数组的指针，继续比较和放入，直到其中一个子数组全部放入结果数组，再将另一个子数组剩余的元素全部放入结果数组。
    

### 6. 堆排序

- **基本思想**：利用堆这种数据结构所设计的一种排序算法。将待排序的数组构造成一个大顶堆（或小顶堆），此时，整个数组的最大值（或最小值）就是堆顶的根节点。将其与末尾元素进行交换，此时末尾就为最大值（或最小值）。然后将剩余的 `n - 1` 个元素重新构造成一个堆，重复上述过程，就能得到一个有序数组。
- **操作步骤**：
    
    - **构建堆**：将数组构建成一个大顶堆（或小顶堆）。从最后一个非叶子节点开始，依次对每个非叶子节点进行调整，使其满足堆的性质（父节点大于或等于子节点，对于大顶堆）。
    - **交换与调整**：将堆顶元素与堆的最后一个元素交换位置，此时堆的大小减 1。然后对新的堆顶元素进行调整，使其重新满足堆的性质。
    - 重复上述交换与调整步骤，直到堆的大小为 1，此时数组即为有序数组。
    

### 7. C++ 标准库排序

- **`std::sort`**：这是 C++ 标准库提供的通用排序函数，定义在 `<algorithm>` 头文件中。它采用了一种混合排序算法（通常是快速排序的变体），具有较高的效率。
    - **函数原型**：`std::sort(begin, end);`，其中 `begin` 是指向要排序范围起始位置的迭代器，`end` 是指向要排序范围末尾位置之后一个位置的迭代器。它默认对范围内的元素进行升序排序。
    - **自定义比较函数**：`std::sort(begin, end, compare);`，`compare` 是一个自定义的比较函数，用于定义排序的规则。例如，要进行降序排序，可以定义一个比较函数，使得当第一个参数大于第二个参数时返回 `true`。
在 Python 中，排序操作可通过多种方式实现，以下是常见排序框架的操作：

### 1. 内置函数 `sorted()`

- **功能**：对可迭代对象进行排序，返回一个新的已排序列表，原可迭代对象不变。
- **参数**：
    
    - `iterable`：必填，需要排序的可迭代对象，如列表、元组、集合等。
    - `key`：可选，是一个函数，用于指定排序的依据。该函数接收可迭代对象中的一个元素作为参数，返回一个用于比较的值。例如，对字典列表按照字典中某个键的值进行排序。
    - `reverse`：可选，布尔值，默认为 `False`，表示升序排序；若为 `True`，则进行降序排序。
    

### 2. 列表的 `sort()` 方法

- **功能**：对列表进行原地排序，即直接修改原列表，使其元素按指定顺序排列，无返回值。
- **参数**：
    - `key`：可选，与 `sorted()` 中的 `key` 作用相同，用于指定排序依据。
    - `reverse`：可选，与 `sorted()` 中的 `reverse` 作用相同，布尔值，控制升序或降序排序。
