在 C++ 中，迭代器是一种用于遍历容器元素的对象，它提供了一种统一的方式来访问容器中的数据，而无需关心容器的具体实现。以下是迭代器框架的常见操作：

### 1. 迭代器类型

- **输入迭代器**：
    
    - 支持通过 `*` 运算符读取其所指向的元素。
    - 支持通过 `++` 运算符（前置和后置）移动到下一个元素。
    - 支持使用 `==` 和 `!=` 运算符进行比较，以判断两个迭代器是否指向相同位置。
    
- **输出迭代器**：
    
    - 支持通过 `*` 运算符写入其所指向的位置。
    - 支持通过 `++` 运算符（前置和后置）移动到下一个位置。
    
- **前向迭代器**：
    
    - 具有输入迭代器和输出迭代器的所有功能。
    - 可以多次读取和写入同一位置。
    
- **双向迭代器**：
    
    - 在前向迭代器的基础上，还支持通过 `--` 运算符（前置和后置）向后移动到前一个元素。
    
- **随机访问迭代器**：
    
    - 具有双向迭代器的所有功能。
    - 支持使用 `+`、`-` 运算符进行迭代器的算术运算，用于直接跳转到指定位置。例如，`it + n` 表示将迭代器 `it` 向前移动 `n` 个位置，`it - n` 表示向后移动 `n` 个位置。
    - 支持使用 `<`、`>`、`<=`、`>=` 运算符进行比较，以判断迭代器的相对位置。
    - 支持使用 `[]` 运算符进行随机访问，例如 `it[n]` 等价于 `*(it + n)`。
    

### 2. 迭代器操作

- **解引用**：使用 `*` 运算符获取迭代器所指向的元素。例如，对于迭代器 `it`，`*it` 返回其所指向的元素。对于指向对象的迭代器，还可以使用 `->` 运算符来访问对象的成员，如 `it->member` 等价于 `(*it).member`。
- **移动迭代器**：
    
    - **前置递增**：使用 `++it` 将迭代器移动到下一个位置，并返回移动后的迭代器引用。常用于循环中，如 `for (auto it = container.begin(); it != container.end(); ++it)`。
    - **后置递增**：使用 `it++` 将迭代器移动到下一个位置，但返回移动前的迭代器副本。通常在不需要立即使用移动后迭代器的场景下使用。
    - **前置递减**：仅适用于双向迭代器和随机访问迭代器，`--it` 将迭代器移动到前一个位置，并返回移动后的迭代器引用。
    - **后置递减**：同样仅适用于双向迭代器和随机访问迭代器，`it--` 将迭代器移动到前一个位置，并返回移动前的迭代器副本。
    - **随机访问移动**：仅适用于随机访问迭代器，通过 `it + n` 或 `it - n` 将迭代器向前或向后移动 `n` 个位置，返回移动后的迭代器。
    
- **比较迭代器**：
    
    - **相等性比较**：使用 `==` 和 `!=` 运算符判断两个迭代器是否指向相同位置。这在遍历容器时用于确定是否到达容器末尾，如 `while (it != container.end())`。
    - **顺序比较**：仅适用于随机访问迭代器，使用 `<`、`>`、`<=`、`>=` 运算符判断两个迭代器的相对位置。例如，`it1 < it2` 判断 `it1` 是否在 `it2` 之前。
    
- **迭代器运算**：
    
    - **加法运算**：随机访问迭代器支持 `it + n`，返回一个新的迭代器，指向当前位置向前移动 `n` 个元素后的位置。
    - **减法运算**：随机访问迭代器支持 `it - n`，返回一个新的迭代器，指向当前位置向后移动 `n` 个元素后的位置。还支持两个随机访问迭代器相减，如 `it2 - it1`，返回它们之间的距离（元素个数）。
    
- **获取容器特定迭代器**：
    
    - **`begin()` 和 `end()`**：容器类通常提供 `begin()` 成员函数返回指向容器第一个元素的迭代器，`end()` 成员函数返回指向容器最后一个元素之后位置的迭代器（常被称为 “超尾” 迭代器）。这两个迭代器用于定义遍历范围，如 `for (auto it = container.begin(); it != container.end(); ++it)`。
    - **`rbegin()` 和 `rend()`**：一些容器（如 `std::vector`、`std::list` 等）提供这两个成员函数，`rbegin()` 返回指向容器最后一个元素的反向迭代器，`rend()` 返回指向容器第一个元素之前位置的反向迭代器。用于反向遍历容器，如 `for (auto it = container.rbegin(); it != container.rend(); ++it)`。
    - **`cbegin()` 和 `cend()`**：与 `begin()` 和 `end()` 类似，但返回的是常量迭代器，不能用于修改容器元素。常用于只读遍历，如 `for (auto it = container.cbegin(); it != container.cend(); ++it)`。
    - **`crbegin()` 和 `crend()`**：与 `rbegin()` 和 `rend()` 类似，但返回的是常量反向迭代器，用于只读反向遍历容器。
    

### 3. 迭代器与算法

C++ 标准库中的许多算法都依赖迭代器来操作容器。这些算法通过迭代器获取容器元素，并在不依赖具体容器类型的情况下进行各种操作。例如：

- **排序算法**：如 `std::sort`，接受一对迭代器，表示要排序的范围，对该范围内的元素进行排序。例如 `std::sort(container.begin(), container.end())`。
- **查找算法**：如 `std::find`，接受一对迭代器和要查找的值，在指定范围内查找该值，并返回指向找到元素的迭代器，如果未找到则返回结束迭代器。例如 `auto it = std::find(container.begin(), container.end(), value)`。
- **遍历算法**：如 `std::for_each`，接受一对迭代器和一个函数对象（或可调用对象），对范围内的每个元素应用该函数对象。例如 `std::for_each(container.begin(), container.end(), [](auto& element) { /* 对 element 进行操作 */ })`。
