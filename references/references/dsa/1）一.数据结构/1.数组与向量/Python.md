### List 框架操作

1. **创建与初始化**
    
    - 直接使用方括号 `[]` 初始化，可包含不同类型元素。
    - 使用 `list()` 函数将可迭代对象转换为列表。
    
2. **访问元素**
    
    - 通过索引访问单个元素，索引从 `0` 开始，支持负数索引。
    - 使用切片操作 `[start:stop:step]` 获取子列表，`start` 为起始索引（包含），`stop` 为结束索引（不包含），`step` 为步长。
    
3. **修改元素**
    
    - 直接通过索引赋值修改指定位置元素。
    
4. **增加元素**
    
    - `append()`：在列表末尾添加单个元素。
    - `extend()`：将一个可迭代对象的所有元素添加到列表末尾。
    - `insert(index, element)`：在指定索引位置插入元素。
    
5. **删除元素**
    
    - `pop([index])`：移除并返回指定索引位置的元素，若不指定索引则移除并返回最后一个元素。
    - `remove(element)`：移除列表中第一个匹配的元素，若元素不存在则报错。
    - `del list[index]`：删除指定索引位置的元素，也可使用切片形式删除多个元素。
    
6. **查找与计数**
    
    - `index(element[, start[, end]])`：返回指定元素首次出现的索引位置，可指定搜索范围，若元素不存在则报错。
    - `count(element)`：统计指定元素在列表中出现的次数。
    
7. **排序与反转**
    
    - `sort(key=None, reverse=False)`：对列表进行原地排序，可通过 `key` 函数指定排序依据，`reverse` 控制升序或降序。
    - `sorted()`：返回一个新的已排序列表，原列表不变。
    - `reverse()`：对列表进行原地反转。
    
8. **其他操作**
    
    - `len()`：返回列表元素个数。
    - 使用 `in` 和 `not in` 运算符检查元素是否在列表中。
    - 列表可通过 `+` 运算符进行拼接，通过 `*` 运算符进行重复。
    

### numpy 数组操作

1. **创建与初始化**
    
    - `numpy.array()`：将 Python 列表、元组等转换为 numpy 数组。
    - `numpy.zeros(shape)`：创建指定形状的全零数组。
    - `numpy.ones(shape)`：创建指定形状的全一数组。
    - `numpy.empty(shape)`：创建指定形状的未初始化数组。
    - `numpy.arange(start, stop, step)`：创建一个从 `start` 开始，到 `stop` 结束（不包含），步长为 `step` 的数组。
    - `numpy.linspace(start, stop, num)`：创建一个在 `start` 和 `stop` 之间均匀分布 `num` 个元素的数组。
    
2. **数组属性**
    
    - `ndim`：返回数组的维度数。
    - `shape`：返回一个表示数组各维度大小的元组。
    - `size`：返回数组元素的总数。
    - `dtype`：返回数组元素的数据类型。
    
3. **访问元素**
    
    - 通过索引访问单个元素，索引从 `0` 开始，支持负数索引，多维数组需按维度顺序提供索引。
    - 使用切片操作获取子数组，各维度切片操作类似列表，支持省略号 `...` 表示多个冒号。
    
4. **修改元素**
    
    - 直接通过索引赋值修改指定位置元素。
    - 对数组进行数学运算（如 `+`, `-`, `*`, `/` 等）会按元素对数组进行操作，结果会修改原数组（若操作符支持）或生成新数组。
    
5. **数组变形**
    
    - `reshape(shape)`：在不改变数据的情况下改变数组的形状。
    - `ravel()`：将多维数组展平为一维数组。
    - `transpose()`：对数组进行转置，改变维度顺序。
    
6. **增加与删除维度**
    
    - `numpy.expand_dims(arr, axis)`：在指定轴位置增加一个新维度。
    - `numpy.squeeze(arr)`：从数组的形状中删除单维度条目。
    
7. **数组拼接与拆分**
    
    - `numpy.concatenate((a1, a2,...), axis=0)`：沿指定轴连接数组序列。
    - `numpy.vstack((a1, a2,...))`：按垂直方向（行）堆叠数组。
    - `numpy.hstack((a1, a2,...))`：按水平方向（列）堆叠数组。
    - `numpy.split(ary, indices_or_sections, axis=0)`：沿指定轴将数组拆分为多个子数组。
    
8. **数学运算**
    
    - 支持基本的算术运算（`+`, `-`, `*`, `/`, `**` 等）按元素进行计算。
    - `numpy.sum()`：计算数组元素的总和，可指定轴进行计算。
    - `numpy.mean()`：计算数组元素的平均值，可指定轴。
    - `numpy.std()`：计算数组元素的标准差，可指定轴。
    - `numpy.min()` 和 `numpy.max()`：分别返回数组中的最小值和最大值，可指定轴。
    
9. **查找与排序**
    - `numpy.where(condition, [x, y])`：根据条件返回满足条件的元素位置，或根据 `x` 和 `y` 返回相应位置的元素。
    - `numpy.sort()`：对数组进行排序，返回排序后的新数组，原数组不变，可指定轴。
    - `argsort()`：返回数组排序后的索引数组，可指定轴。
