# 一、口述理论题（5 题 ×5 分 = 25 分）

2. 为什么在生产环境中，强烈不建议用 `pd.get_dummies` 做特征编码？应该用什么替代？

4. 数据治理中**主键唯一性**、**非空约束**、**值域约束**、**逻辑一致性**分别指什么？各举一例。
5. 简述 Pipeline 在数据中台特征工程中的作用，为什么必须用 Pipeline 上线模型？

---

## 参考答案

1. **数据血缘**：记录数据从源头→清洗→聚合→指标→应用的全链路流向。
    
    Python 实现：解析 SQL、解析 Pandas 列依赖、记录表 / 字段级输入输出，生成依赖图。
    
2. `get_dummies` 无状态，新数据出现新类别会导致特征维度不匹配，线上直接报错。
    
    替代：`OneHotEncoder`/`OrdinalEncoder`，保存 `encoder` 用于推理。
    
3. 常见问题：
    
    - 复制全表（`df = df.copy()`）
    - 不指定 dtype，默认 int64/float64
    - 大量 apply 导致中间对象
    - 一次性加载所有列
        
        优化：指定类型、分块、只加载需要列、使用 query/eval。
    
4. - 主键唯一：user_id 不重复
    - 非空约束：订单金额不能为空
    - 值域约束：年龄 16–70，金额 ≥0
    - 逻辑一致性：支付时间 > 注册时间
    
5. Pipeline 将编码、缩放、特征选择、模型打包成单一对象，保证训练与线上推理**特征处理逻辑完全一致**，避免线上线下特征不一致导致模型失效。

# 二、NumPy 数值计算（1 题 ×10 分 = 10 分）

给定数组 shape=(500, 8)，正态分布随机数：

1. 对每一列做 **稳健标准化**（中位数 + MAD）
2. 把 |Z| > 3 的值标记为异常并替换为列中位数
3. 计算行与行之间的欧氏距离矩阵（前 10 行）

## 参考答案

```python
import numpy as np

np.random.seed(42)
X = np.random.randn(500, 8)

# 1 稳健标准化
median = np.median(X, axis=0)
mad = np.median(np.abs(X - median), axis=0)
X_rob = (X - median) / (mad + 1e-8)

# 2 异常值替换
for i in range(8):
    col = X_rob[:, i]
    mask = np.abs(col) > 3
    X_rob[mask, i] = np.median(col[~mask])

# 3 前10行欧氏距离
A = X_rob[:10]
dist = np.sqrt(np.sum((A[:, None] - A[None, :])**2, axis=-1))
```

# 三、Pandas 数据质量治理（1 题 ×20 分 = 20 分）

**场景**：用户宽表质量监控（真实中台常用）

df 包含：user_id, phone, id_card, register_time, pay_amt, city, channel

要求：

1. 计算**字段级缺失率、重复率、唯一值个数**，输出质量报告
2. 手机号必须为 1 开头 11 位数字，否则标记 `phone_invalid=1`
3. 身份证非 18 位标记异常，且校验**最后一位校验码**（简单实现即可）
4. pay_amt <0 或> 100000 标记为异常并设为同 channel 中位数
5. 按 user_id 去重，保留 register_time 最大的一条
6. 最终输出 `data_quality_score`（0–100），违反规则一条扣 20 分

## 参考答案（精简可直接写）

```python
import pandas as pd
import numpy as np

df = pd.DataFrame({
    'user_id': [1,1,2,3,4,5],
    'phone': ['13800138000','13800138000','12345','13999999999','abc','13611112222'],
    'id_card': ['110101199001011234','11010119900101123','','110101199001011235','',''],
    'register_time': pd.date_range('2024-01-01', periods=6),
    'pay_amt': [100,100,-50,200000,500,800],
    'city': ['bj','bj','sh','gz','sz','sh'],
    'channel': ['ad','ad','natural','ad','natural','natural']
})

# 1 质量报告
quality = pd.DataFrame({
    'missing': df.isnull().mean(),
    'unique': df.nunique(),
    'duplicate_rate': df.duplicated().mean()
})

# 2 手机号校验
df['phone'] = df['phone'].astype(str).str.replace(r'\D', '', regex=True)
df['phone_invalid'] = ~df['phone'].str.match(r'^1\d{10}$').fillna(False)

# 3 身份证简单校验
df['id_invalid'] = ~df['id_card'].str.match(r'^\d{18}$').fillna(True)

# 4 金额异常处理
channel_median = df.groupby('channel')['pay_amt'].transform(lambda x: x[x.between(0,1e5)].median())
df['pay_amt'] = df['pay_amt'].where(df['pay_amt'].between(0,1e5), channel_median)

# 5 去重
df = df.sort_values('register_time').drop_duplicates('user_id', keep='last')

# 6 质量分
df['error_cnt'] = df[['phone_invalid','id_invalid']].sum(axis=1)
df['data_quality_score'] = (1 - df['error_cnt']*0.2).clip(0,1)*100
```

# 四、数据中台指标宽表构建（1 题 ×18 分 = 18 分）

三张表：

- user：user_id, register_dt, channel
- order：user_id, order_id, pay_amt, pay_dt
- behavior：user_id, pv, cart, fav, dt

要求：

1. 左连接用户全量数据，构建用户宽表
2. 计算用户指标：
    
    - 订单数、总金额、首付时间、末次付时间、支付天数
    - 行为总 pv、加购率 cart/pv
    
3. 构造时间特征：注册到首付间隔天数
4. 按 channel 计算**付费用户占比**与**人均支付金额**

## 参考答案

```python
import pandas as pd

user = pd.DataFrame({
    'user_id': [1,2,3,4,5],
    'register_dt': pd.to_datetime(['2024-01-01']*5),
    'channel': ['ad','ad','natural','ad','natural']
})

order = pd.DataFrame({
    'user_id': [1,1,2,3],
    'pay_amt': [100,200,150,300],
    'pay_dt': pd.date_range('2024-01-02', periods=4)
})

behavior = pd.DataFrame({
    'user_id': [1,2,3,4,5],
    'pv': [20,15,30,5,10],
    'cart': [4,3,6,0,1],
    'fav': [2,1,3,0,0]
})

# 聚合
order_agg = order.groupby('user_id').agg(
    order_cnt=('pay_amt','count'),
    total_amt=('pay_amt','sum'),
    first_pay=('pay_dt','min'),
    last_pay=('pay_dt','max')
)

df = user.merge(order_agg, on='user_id', how='left')
df = df.merge(behavior, on='user_id', how='left')

# 特征
df['pay_days'] = df.groupby('user_id')['first_pay'].transform('nunique').fillna(0)
df['register2pay'] = (df['first_pay'] - df['register_dt']).dt.days.fillna(-1)
df['cart_rate'] = df['cart'] / df['pv'].clip(1)

# 渠道指标
channel_stat = df.groupby('channel').agg(
    pay_rate=('order_cnt', lambda x: (x>0).mean()),
    avg_amt=('total_amt','mean')
)
```

# 五、特征工程 + 建模上线规范（1 题 ×12 分 = 12 分）

要求：

1. 类别特征：低基数 OneHot，高基数 LabelEncoder
2. 数值特征：标准化
3. 使用 Pipeline 封装全过程
4. 分层划分数据集，输出模型可直接训练的 X_train/X_test

## 参考答案

```python
import pandas as pd
from sklearn.compose import ColumnTransformer
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler, OneHotEncoder, LabelEncoder
from sklearn.model_selection import train_test_split

df = pd.DataFrame({
    'age': [22,31,28,35,40],
    'salary': [5000,8000,7500,12000,14000],
    'city': ['bj','sh','bj','gz','sh'],
    'channel': ['ad','natural','ad','ad','natural'],
    'churn': [0,1,0,1,0]
})

X = df.drop('churn', axis=1)
y = df['churn']

num_cols = ['age','salary']
cat_cols = ['city','channel']

preprocessor = ColumnTransformer([
    ('num', StandardScaler(), num_cols),
    ('cat', OneHotEncoder(drop='first'), cat_cols)
])

pipe = Pipeline([
    ('preprocess', preprocessor)
])

X_processed = pipe.fit_transform(X)
X_train, X_test, y_train, y_test = train_test_split(
    X_processed, y, test_size=0.2, random_state=42, stratify=y
)
```

# 六、综合场景题（1 题 ×15 分 = 15 分）

**问题**：

你们数据中台每天产出用户宽表，但下游反馈：

指标波动大、重复用户多、缺失率高、部分用户金额为负。

请你设计一套**自动化数据质量监控 + 自愈**体系，用 Python 实现核心逻辑。

## 参考答案（面试高分回答）

### 架构思路

1. **规则配置化**：主键唯一、非空、值域、逻辑一致性。
2. **自动巡检**：每日调度执行质量扫描。
3. **异常告警**：输出异常明细、质量分数、趋势图。
4. **自愈能力**：可自动修复的脏数据自动清洗，不可修复的拦截并人工处理。
5. **质量看板**：按天 / 表 / 字段统计质量趋势。

### 核心代码框架

```python
def quality_monitor(df):
    # 规则
    rules = {
        'user_id': {'unique': True, 'not_null': True},
        'pay_amt': {'min':0},
        'phone': {'regex': r'^1\d{10}$'}
    }
    report = []
    # 检查
    for col, rule in rules.items():
        if rule.get('not_null'):
            null_pct = df[col].isnull().mean()
            report.append({'col':col, 'null_pct':null_pct})
    # 异常数据
    bad = df[~df['phone'].str.match(r'^1\d{10}$') | df['pay_amt']<0]
    bad.to_csv('bad_data.csv', index=False)
    return pd.DataFrame(report)

# 自愈：简单修复
df['pay_amt'] = df['pay_amt'].clip(lower=0)
```
