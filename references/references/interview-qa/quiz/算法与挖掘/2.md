## 1. 手写 / 口述思路：PCA 的核心流程是什么？为什么要先标准化？协方差矩阵的意义是什么？

### **参考答案**

1. **PCA 流程**
    
    1）对原始特征做**标准化**（均值 0、方差 1）；
    
    2）计算**协方差矩阵**；
    
    3）对协方差矩阵**特征值分解**，得到特征值与特征向量；
    
    4）按特征值从大到小排序，取前 k 个特征向量构成投影矩阵；
    
    5）将原始数据投影得到降维后结果。
    
2. **为什么要标准化**
    
    量纲不同的特征（如收入万元、年龄几十）会直接主导方差，PCA 会被大数特征 “带偏”，标准化后各特征方差可比。
    
3. **协方差矩阵意义**
    
    描述特征间**线性相关程度**，PCA 本质是找一组新基，使投影后**方差最大、冗余最小**。

## 2. XGBoost 为什么要用二阶泰勒展开？与传统 GBDT 区别在哪？XGBoost 如何处理缺失值？

### **参考答案**

1. **二阶泰勒展开**
    
    用一阶（梯度）+ 二阶（海森）信息，能更精准逼近损失函数，提升收敛速度与稳定性，便于自定义损失与正则。
    
2. **与 GBDT 区别**
    
    - GBDT 仅用一阶梯度，XGBoost 用二阶；
    - XGBoost 内置 L1/L2 正则，防过拟合；
    - 支持列抽样、行抽样；
    - 并行优化（特征粒度）；
    - 自带缺失值自动学习分裂方向。
    
3. **缺失值处理**
    
    训练时自动把缺失值分别划入左子树 / 右子树，选择**增益更大**的方向，预测时沿用该策略。

## 4. 聚类场景题：用户行为 Embedding 高维稀疏，用 K-Means 效果差怎么办？说出至少 3 种优化方案

### **参考答案**

1. **先降维**：PCA/TSNE/UMAP 把 Embedding 降到 32~128 维，再聚类。
2. **换算法**：改用 DBSCAN/HDBSCAN，避免指定 K，更适合密度不均、非凸簇。
3. **距离度量替换**：K-Means 默认欧氏距离，高维空间失效；改用余弦相似度 / 皮尔逊相关系数。
4. **预处理**：对 Embedding 做归一化，去除极端异常点。
5. **层次聚类 + 剪枝**：小批量先做层次聚类确定大致簇数，再用 K-Means 加速。

## 5. RAG 全链路优化题：检索召回不精准、LLM 出现幻觉，分别怎么解决？请给出工程化方案

### **参考答案**

1. **召回不精准**
    
    - 混合召回：BM25 + 向量检索（互补）
    - 文档分块优化：固定长度 + 滑动窗口，保留上下文
    - 向量模型选型：用领域微调 Embedding 模型
    - 关键词增强：对 Query 做实体抽取、关键词加权
    
2. **幻觉优化**
    
    - 增强重排阶段：CrossEncoder / BERT 重排，提升 top-k 相关性
    - 提示词工程：强制 LLM 只依据参考内容回答，禁止编造
    - 引用校验：让模型标注引用片段，后处理校验来源
    - 加入事实校验小模型 / 规则过滤
    
3. **工程链路**
    
    切分 → 向量化 → 索引 → 多路召回 → 重排 → 上下文截断 → LLM 生成 → 后校验
