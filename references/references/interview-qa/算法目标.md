# RAG 技术体系（重点！写这个要真懂）

## 必须掌握的完整链路

1. **文档加载与分块**：PDF/Word/Txt/ 网页解析，chunk 切分
2. **向量化**：Embedding 模型生成向量
3. **向量库存储**：Milvus/FAISS/Chroma 存储与检索
4. **召回阶段**：
    
    - 向量检索（语义召回）
    - BM25 关键词召回（精确匹配）
    - 多路召回 + 重排
    
5. **提示词工程**：将召回内容塞进 prompt，避免幻觉
6. **生成回答**：调用 LLM（通义千问、文心、Llama、Qwen 等）
7. **效果优化**：
    
    - 分块大小优化
    - 检索召回率优化
    - 上下文窗口管理
    - hallucination 控制
