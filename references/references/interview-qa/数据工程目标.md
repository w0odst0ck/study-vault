## 1. 熟练 Python（Pandas/Numpy/Scikit-learn）

### 要达到的水平

- **Python**
    
    - 能独立写脚本、写函数、写类，会异常处理、文件读写、日志、定时任务。
    - 能用 `requests` 爬 / 接口调用，会 `json`/`csv`/`excel` 处理。
    
- **Pandas**（核心中的核心）
    
    - 数据读取、清洗、缺失值 / 异常值处理
    - 分组聚合 `groupby`、透视表 `pivot_table`
    - 多表合并 `merge/concat/join`
    - 时间序列处理、字段转换、批量处理逻辑
    - 能写出**可读性强、可复用**的数据处理 pipeline
    
- **Numpy**
    
    - 数组操作、向量化计算、矩阵运算
    - 会用常用函数，不用死记，但遇到能立刻写出来
    
- **Scikit-learn**
    
    - 数据预处理：标准化、归一化、编码
    - 分类 / 回归模型训练、预测、评估指标
    - train-test split、交叉验证、简单模型调参
    - 能做**特征工程 + baseline 模型**，不是只会调包
    

### 一句话标准

> 给你一张脏表，你能**独立清洗、加工、分析、输出结果**，并能简单建模。

## 2. ETL 流程搭建与自动化

### 要达到的水平

- 懂 ETL 含义：**抽取 Extract → 转换 Transform → 加载 Load**
- 能写完整 ETL 脚本：
    
    - 从 MySQL/PostgreSQL/Excel/CSV 抽数
    - 清洗、字段映射、关联、去重、计算指标
    - 写入目标库（MySQL/Hive/ClickHouse 等）
    
- 会**自动化**：
    
    - Linux crontab 定时
    - 或 Airflow 调度（有就加分）
    - 日志、错误重试、监控告警
    

### 一句话标准

> 能独立设计并落地**可定时运行、稳定可靠**的数据同步 / 加工任务。
