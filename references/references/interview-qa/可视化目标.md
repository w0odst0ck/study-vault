# 熟练 Matplotlib / Seaborn / Streamlit

## 1. Matplotlib（基础绘图）

你需要做到：

- 画**常用图表**：折线、柱状、条形、散点、饼图、直方图、热力图
- 能控制图表细节：标题、坐标轴、图例、颜色、字体、尺寸、子图
- 能把 Pandas 数据**直接可视化**，不用复杂处理
- 能输出高清图片（PNG/PDF），用于报告 / PPT

一句话底线：

**给你一份清洗好的数据，你能快速画出专业、干净、可解释的图表。**

---

## 2. Seaborn（统计可视化）

你需要做到：

- 比 Matplotlib 更美观、更快速地画**统计类图表**
- 熟练：箱线图、小提琴图、分布图、相关性热力图、多子图网格
- 能做**数据探索分析 EDA**：分布、异常、相关性、分组对比

一句话底线：

**你能快速做数据探索，画出专业、美观、适合汇报的统计图表。**

---

## 3. Streamlit（核心技能！交互式看板）

这是你写 “**可独立开发 BI 看板、数据分析平台**” 的关键，必须真会。

### 必须掌握：

- 10 分钟内能搭出一个**可运行的数据分析网页**
- 组件使用：侧边栏、下拉框、日期选择、滑块、多选、按钮、上传文件
- 实时读取数据：CSV / Excel / MySQL / 数据接口
- 展示 Pandas 表格、Matplotlib/Seaborn 图表、Plotly 图表
- 页面布局：多列、卡片、分段、标题
- 简单的数据筛选、聚合、计算、导出功能

### 进阶（写 “看板 / 平台” 必须会）

- 多页面应用（Multipage Apps）
- 缓存数据提升速度
- 连接数据库做**实时数据看板**
- 简单权限 / 密码访问
- 部署上线（本地 / 服务器 / 云）

一句话底线：

**不用前端知识，纯 Python 就能快速做出可给业务 / 领导使用的交互式数据看板。**

---

# 可独立开发交互式 BI 看板、多维数据分析平台

这句不是吹牛，是**真实可落地的能力**，要求：

## 1. 交互式 BI 看板（必须做到）

- 支持**筛选条件**：日期、部门、产品、地区
- 支持**图表联动**：选一个时间，所有图自动刷新
- 包含核心指标：总金额、总单量、转化率、同比环比、排行榜
- 图表类型：折线趋势、柱状对比、饼图占比、表格明细
- 自动刷新、可搜索、可导出

## 2. 多维数据分析平台（比看板更高级）

- 支持**下钻分析**：总览 → 按地区 → 按店铺 → 按商品
- 支持**灵活聚合**：按日 / 周 / 月 / 年、按维度组合统计
- 支持**自定义查询**：用户自己选维度、指标
- 支持**数据导出、打印、分享链接**

一句话底线：

**你能一个人完成：数据读取 → 清洗 → 可视化 → 交互看板 → 上线使用。**
